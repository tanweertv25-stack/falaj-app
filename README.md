# نظام إدارة الفلج

تطبيق عربي لإدارة جدول الردات ومتابعة حالة السقي الحالية والقادمة، مع لوحة خاصة للإدارة.

## الميزات

- عرض الحالة الحالية للفلج والوقت المتبقي.
- عرض الردة التالية والجدول القادم.
- بحث شخصي بالاسم أو رقم المالك.
- لوحة إدارة لإضافة وتعديل وحذف الردات.
- حساب اليوم تلقائيًا عند اختيار التاريخ.
- تنبيهات للتعارضات أو الفواصل القصيرة بين الردات.
- سجل بلاغات ومهام صيانة مع أولوية وحالة متابعة.
- تصدير الجدول بصيغة CSV.
- تصدير واستيراد نسخة احتياطية JSON.
- زر واتساب عائم للتواصل مع المشرف على الرقم `+96896017822`.
- تصميم متجاوب يعمل على الجوال والتابلت وسطح المكتب.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح الرابط الذي يظهر في الطرفية.

## البناء

```bash
npm run build
```

كلمة مرور الإدارة الحالية موجودة داخل `src/App.jsx` في المتغير `ADMIN_PASSWORD`.
